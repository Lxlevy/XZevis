# XZevis
 
A stupid lil fun website made by [Lxlevy](https://github.com/Lxlevy) just for the fun of it.
 
## Features
 
### Tools
Offers a variety of tools and experiences embedded directly in the browser, including virtual machines running inside iframes, JSNES (a JavaScript NES emulator), and more.
 
### Themes
Comes with a bunch of themes to customize the look and feel of the site to your liking.
